# AI-Based Fake News Generator & Detector

An AI-powered web application that can:

- Generate fake news headlines/articles using prebuilt NLP models
- Detect whether a news article is Fake or Real
- Provide a simple and interactive user interface using Streamlit

---

# Features

- Fake News Generation
- Fake News Detection
- Simple and responsive UI
- Uses pretrained AI/NLP models
- Beginner-friendly project structure

---

# Technologies Used

- Python
- Streamlit
- Hugging Face Transformers

# Installation

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/fake-news-project.git
cd fake-news-project
```

## 2. Create Virtual Environment (Optional)

```bash
python -m venv venv
```

## 3. Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Mac/Linux

```bash
source venv/bin/activate
```

## 4. Install Dependencies

```bash
pip install -r requirements.txt
```

---

# Run the Application

```bash
streamlit run app.py
```

---

# How It Works

## Fake News Generator

- User enters a topic or headline
- AI model generates fake news content

## Fake News Detector

- User enters a news article or headline
- AI model predicts whether the news is:
  - Fake
  - Real
